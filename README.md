### Customizing Authentication

Authentication is managed by src/lib/AuthContext.jsx. It currently
provides a basic stub that always considers the user authenticated. To
integrate Clerk (or another provider) replace the contents of that file with
calls to your chosen SDK. Your Clerk keys can be read from environment
variables as shown above.
